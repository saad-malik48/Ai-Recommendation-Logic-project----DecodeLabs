# AI Recommendation Logic

A simple recommendation system built in Python that takes user interests as input and returns personalized item suggestions using keyword matching logic.

---

## About the Project

This project was built as part of my **Week 3 task during my internship at DecodeLabs**. The goal was to understand the core concepts behind recommendation systems — how to capture user preferences, match them against a dataset, and return relevant results.

---

## How It Works

1. The program asks the user to enter their interests
2. It matches those interests against a built-in dataset of items
3. It scores each item based on keyword similarity
4. It displays the top 3 recommendations with match reasons

---

## Features

- Takes user input at runtime via the terminal
- Keyword-based preference matching
- Ranked recommendations with reasons
- Clean and readable output
- No external libraries required

---

## Getting Started

### Requirements

- Python 3.x
- No installations needed — uses standard library only

### Run the Project

```bash
python recommend.py
```

```
Enter your interests: python, AI, data

Your top recommendations:
1. Python for Beginners Course  — matches: python
2. AI Fundamentals Guide        — matches: AI
3. Data Science Handbook        — matches: data
```
